// learning topics

いま学んでいること

01

CSS レイアウト

Flexbox と Grid の使い分け。レスポンシブデザインの基礎から応用まで。

02

タイポグラフィ

フォントの選び方・組み合わせ方・読みやすさを決める行間と文字間隔。

03

カラー理論

色相・彩度・明度の関係。補色・類似色を使ったパレット設計の考え方。

04

UI コンポーネント

ボタン・フォーム・カード。再利用できるパーツとしての設計と実装。

05

アニメーション

CSS transition と keyframes。「使いすぎない」ための判断基準を学ぶ。

06

デザインシステム

色・フォント・スペーシングをトークン化してサイト全体に一貫性を持たせる。
