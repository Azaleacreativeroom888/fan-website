Blog

2025-01-15

Flexbox の align-items と align-content の違いをようやく理解した。align-items は各行の中での揃え方、align-content は複数行まとめての揃え方。図に描いたらすっきりした。

2025-01-12

CSS カスタムプロパティで色管理が劇的に楽になった。--primary みたいな変数を :root に定義して使いまわすだけで、変更が一箇所で済む。

2025-01-08

font-size に rem を使う理由がわかった。px は固定だけど rem はルート要素に対する相対値だから、ブラウザのフォントサイズ設定を尊重できる。アクセシビリティのために大事。

2025-01-03

Grid の auto-fill と auto-fit、何度読んでも混乱するので図に描いてみた。空のトラックを残す（auto-fill）か潰す（auto-fit）かの違い。minmax と組み合わせると強い。

2024-12-28

position: sticky がなぜ効かないのかしばらく悩んだ。親要素に overflow: hidden がついてると動かない。罠すぎる。

2024-12-22

box-sizing: border-box を全称セレクタで当てるのが当たり前らしい。padding を足してもボックスのサイズが変わらないのは確かに便利。

2024-12-18

clamp() を覚えた。clamp(最小値, 推奨値, 最大値) で流体タイポグラフィが実現できる。vw をそのまま使うと極端になるからこれで制御するのがベター。
