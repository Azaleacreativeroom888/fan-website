Flexbox 完全理解メモ

Flexbox は「一方向のレイアウト」を担う仕組み。横並びにするだけでなく、余白の配分・揃え方・折り返しまで細かく制御できる。混乱しやすいプロパティの違いを整理した。

justify-content は主軸方向（横）の配置。flex-start / center / space-between など。
align-items は交差軸方向（縦）の各行内での揃え方。
align-content は複数行がある場合の行間の配分。1行だけのときは効かない。
flex-grow / flex-shrink / flex-basis は省略形 flex: 1 1 auto でまとめて指定できる。
.container { display: flex; justify-content: space-between; align-items: center; gap: 1rem; }

CSS カスタムプロパティで色を管理する

CSS カスタムプロパティ（変数）を使うと、色・サイズ・フォントなどを一元管理できる。デザイントークンの考え方に近く、変更コストが劇的に下がる。

:root に定義すると全体から参照できるグローバル変数になる。
var(--変数名) で呼び出す。第二引数にフォールバック値を指定できる。
JavaScript からも getPropertyValue / setProperty で読み書きできる。
ダークモードの切り替えも @media (prefers-color-scheme) と組み合わせると楽。
/* 定義 */ :root { --color-primary: #7CB82F; --color-accent: #D63B3B; } /* 使用 */ button { background: var(--color-primary); }

CSS Grid — auto-fill と auto-fit の違い

repeat() の中で使う auto-fill と auto-fit は一見同じに見えるが、空のトラックの扱いが違う。minmax() と組み合わせることで真価を発揮する。

auto-fill：アイテムが少なくても空のトラックを残してカラムを維持する。
auto-fit：空のトラックを折り畳んで、アイテムが伸びて埋める。
どちらも minmax(最小幅, 1fr) と組み合わせるのがセオリー。
アイテムが多いときは両者の見た目は同じになる。
/* アイテムが少ないと挙動が変わる */ .grid-fill { grid-template-columns: repeat(auto-fill, minmax(200px, 1fr)); } .grid-fit { grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); }

流体タイポグラフィ — clamp() の使い方

clamp() を使うと、画面幅に応じてフォントサイズをなめらかに変化させながら、最小・最大の上限下限も同時に設定できる。メディアクエリなしで流体タイポグラフィが実現できる。

構文：clamp(最小値, 推奨値, 最大値)
推奨値に vw を使うと画面幅に比例して変化する。
vw だけだと小さい画面で極端に小さくなるため clamp で制御するのがベスト。
font-size だけでなく padding や gap にも使える。
h1 { /* 最小2rem、画面幅の5%、最大4rem */ font-size: clamp(2rem, 5vw, 4rem); } .section { padding: clamp(2rem, 8vw, 6rem) 1rem; }
